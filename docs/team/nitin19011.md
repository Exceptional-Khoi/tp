### üîí Feature: Robust Validation for Core Commands
- Standardized validation + error handling for:
    - `/create_workout` (date/time format, overlapping session prevention),
    - `/add_exercise`, `/add_set` (workflow order + numeric constraints),
    - `/end_workout` (end time after start time),
    - `/delete_workout` (graceful handling of invalid indices).
- Unified error text conventions across commands for a consistent CLI experience.

**Impact:** Fewer crashes, clearer guidance, and easier testing due to predictable failure modes.
